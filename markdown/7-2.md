## 7.2. 日本語項目と表意定数

日本語項目(PIC N項目)における各表意定数の値は、次の通りである。

表7-1-日本語項目と表意定数の値

| 表意定数 | シフトJISコード |
| --- | :--- |
| SPACE(S) | 日本語空白文字 X"8140" |
| HIGH-VALUE(S) | X"FFFF" |
| LOW-VALUE(S) | X"0000" |
| ALL 定数 | 定数の値に依存する |
